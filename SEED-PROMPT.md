# AI-SDLC: Seed Prompt

> This is for humans. Copy the prompt below and paste it into Cowork (or your AI tool)
> at the start of a new session. Point your AI tool at the folder where you want
> your workspace to live.

---

## Prerequisites

- A folder on your machine where the workspace will live (this is where you point your AI tool)
- The URL of the code repo you want to work on
- SSH access configured if the repo is private

---

## The Prompt

Copy everything inside the block below and paste it into a new session:

```
I'm setting up a new project under the AI-SDLC methodology.

Fetch the bootstrap instructions from:
https://raw.githubusercontent.com/alberto-conan-ui/ai-sdlc/refs/heads/main/BOOTSTRAP.md

Read the full document, then follow its steps.
```

That's it. The AI will fetch the setup instructions, clone the repo, and walk you through the rest.

---

## What Happens Next

1. The AI fetches BOOTSTRAP.md (the entry point)
2. It asks you about your code repo and project
3. It clones the repos and creates the project journal with skeleton files — all automated
4. You review the result

By the end you'll have a three-folder workspace in INCEPTION mode, ready for your first session with the Architect.
